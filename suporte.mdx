## Perguntas frequentes

<AccordionGroup>
  <Accordion title="O portal parou de abrir ou aparece uma mensagem de erro" icon="triangle-exclamation">
    Quase sempre é porque o favorito salvo aponta para um **endereço de login temporário** (longo, com `login`, `oauth` ou `app.snowflake.com`), que expira depois de um tempo.

    A solução é usar o **endereço fixo** do portal. Siga o guia [Salvar o Portal nos Favoritos](/salvar-portal-favoritos) — em poucos passos você corrige o favorito sem precisar apagá-lo.
  </Accordion>

  <Accordion title="Qual é o endereço correto do portal?" icon="link" defaultOpen={true}>
    Use sempre:

    ```
    https://xxxx-xxxx-xxx.snowflakecomputing.app/
    ```

    O endereço certo é **curto**, **sempre o mesmo** e termina em `.snowflakecomputing.app/`. Se o seu favorito for diferente disso, ele está errado.
  </Accordion>

  <Accordion title="Como salvo o portal nos favoritos para não errar de novo?" icon="star">
    Temos um passo a passo para **Google Chrome** e **Microsoft Edge**, tanto para salvar um favorito novo quanto para corrigir um existente. Veja em [Salvar o Portal nos Favoritos](/salvar-portal-favoritos).
  </Accordion>

  <Accordion title="Não consigo fazer login / esqueci minha senha" icon="key">
    Se o endereço está correto mas o acesso falha, fale com o **seu contato Astecha** ou pelo **WhatsApp** acima. Informe o seu usuário (e‑mail) para que possamos verificar o seu acesso.
  </Accordion>
</AccordionGroup>
